# Report using React


SPA demonstrating html report with search.

Using:
  - React
  - Data retreived via Fetch
  - Boostrap


### Installation

Requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ npm install
$ npm start
```
